• Load Height: It is the height of the object expected to be raised.
  • Load Width: it is the width of the object expected to be raised.
  • Load Depth: It is the depth of the object expected to be raised.
  • Sling Height: It is the Height of the sling used to raise the object.
  • Raising Height: The total Height of the Building.
  • Safety Height: Extra Height used for safety purposes.
  • Object Weight: The weight of the object to be raised in tonnage.
  • Sling Weight: The weight of the sling used to raise the object in tonnage.
  • Safety weight: It is calculated so that the entered weight is 85% of total weight.
  • Distance to the building edge: The distance between the crane center and the edge of the building.
  • Radius of work: the distance between crane center and center of location of the object to be installed.
  • Boom Angle: It is the minimum angle between the horizontal plan and the boom to perform the job.
  • Boom Length: it is the minimum boom length needed to perform the job.
